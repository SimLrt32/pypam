.. currentmodule:: pypam

Install
=======

Using pip distribution
----------------------
.. code-block::

    pip install lifewatch-pypam


Using git clone
---------------

1. Clone the package

.. code-block::

    git clone https://github.com/lifewatch/pypam.git


2. Use poetry to install the project dependencies

.. code-block::

    poetry install


3. Build the project

.. code-block::

    poetry build
